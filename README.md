Set it up:

1.  `git clone github.com/2lach/2lach.github.io.git`
2.  `yarn install`
3.  `yarn start`

```npm scripts
   - build
      node scripts/build.js
   - prettier
      prettier --write "./**/*.{js,jsx,json}"
   - start
      node scripts/start.js
   - test
      node scripts/test.js
```
